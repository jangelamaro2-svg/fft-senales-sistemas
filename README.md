# Simulación y análisis de señales con la Transformada de Fourier

Este repositorio contiene el código desarrollado en Python para la simulación y análisis de señales en el dominio del tiempo y de la frecuencia utilizando la Transformada de Fourier (FFT).

## Objetivo

Generar señales básicas (pulso rectangular, función escalón y señal senoidal) y analizar su comportamiento en frecuencia mediante la función `np.fft.fft()`.

## Contenido

- Generación de señales en el dominio del tiempo  
- Cálculo de la Transformada de Fourier  
- Visualización de magnitud del espectro  
- Verificación de propiedades básicas de la transformada  

## Tecnologías utilizadas

- Python  
- NumPy  
- Matplotlib
